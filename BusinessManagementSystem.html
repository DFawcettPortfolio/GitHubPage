Business Management System

Business Management System:

     This project was developed in Visual Studio to integrate with a SQL Server database, enabling businesses to process inventory requests for job orders.

     For demonstration purposes, this version has been adapted to use an embedded SQLite database within the executable. This approach allows the application to run as a self-contained demo without requiring external dependencies.

     A production-ready version can be quickly deployed to connect with a designated SQL Server instance, including modifications to meet individual business needs.

With this program, Administrator-level Users can:

  • Add, remove, and modify Employee information, including login credentials to the program itself
  • Add, remove, and modify Inventory, allowing for easy tracking and control
  • Approve, Deny, or Place orders for inventory requests
  • Directly Query the database
Non-Administrator Users can:
  • Place orders for inventory requests
